# Stonks

## Development

- **Task runner**: `just` — run `just` to see all commands
- **Start dev**: `just dev` (runs backend + frontend in parallel)
- **Database**: `just db-start` to start dev postgres, shared across worktrees

## Worktree Development

Use `claude --worktree <name>` to create a worktree, then run `/worktree setup` to assign ports and install deps. See `/worktree` skill for full details.
